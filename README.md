En este repositorio se encuentra el mismo código que en Colab y la carpeta de entra en el .rar. Para correr el programa debe:
Clonar el repositorio
$ git clone https://github.com/ezequieljm/Fourier-Serie-Expansions-App

Dirigirse a la carpeta \Fourier-Serie-Expansions-App y crear un entorno virtual y activarlo
$ viratualenv env
$ .\env\Scripts\activate

Descargar las dependencias.
$ pip install -r requirements.txt

Luego debe correr el archivo main.py
	$ python .\src\main.py
